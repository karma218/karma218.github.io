Projects

Autonomous Vehicle Lab, Cal Poly Pomona

This project utilizes ROS2 Humble with FastDDS middleware for managing an autonomous vehicle. It includes an Nvidia AGX Orin for high-level robotics tasks and sensor data collection, a Raspberry Pi for communication with an Arduino Mega for control and odometry, and for interfacing with two USB cameras. Each directory in the repository's root represents code for a specific device.

UAV Lab, Cal Poly Pomona

The project aims to develop collaborative mission execution capabilities using multiple UAVs for fire detection and suppression. This includes developing a custom machine learning model for fire detection, integrating a UAV with a fire suppressant dropping mechanism, and creating a Ground Control Software for command and control.
